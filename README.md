# Restaurant Management System

## Project Description
This project is a simple Restaurant Management System developed using Java.
It models products, dishes, beverages, billing, and stock control using object-oriented programming principles.

## Features
- Product management
- Dish and beverage modeling
- Billing system
- Stock availability control
- Custom exception handling

## Technologies Used
- Java
- IntelliJ IDEA
- Git & GitHub

## Project Structure

## How to Run
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Run `Main.java`
